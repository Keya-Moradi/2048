How to Play 2048

Welcome to the 2048 game! Your goal is to combine tiles with the same number to create a tile with the value of 2048.

Game Setup

  • The game starts with a 4x4 grid, initially containing two tiles, each showing a number (either 2 or 4).
  • New tiles will appear after every move you make, in a random empty position on the grid.

How to Play

  1. Objective: Slide numbered tiles on a grid to combine them and create a tile with the number 2048.
  2. Controls: Use the arrow keys on your keyboard to move tiles in the corresponding direction (up, down, left, right).
    Note: Only the arrow keys are functional for gameplay; no other keys, such as the space bar, affect the game.
  3. Merging Tiles: When two tiles with the same number collide while moving, they merge into one tile with a value equal to their sum (e.g., two tiles with '2' merge to become a '4'). Each successful merge increases your score by the value of the merged tile.
  4. Game Progression: After every move, a new tile with a value of either 2 or 4 will randomly appear on an empty spot on the grid. The game ends if there are no empty spaces left on the grid and no possible merges.
  5. Winning the Game: The goal is to create a tile with the number 2048 through merging. When you achieve a tile of 2048, a "You Win!" message will appear.
  6. Game Over: If the grid is full and no more merges can be made, the game will display a "Game Over!" message.

Additional Features

  • You can restart the game at any time by clicking the Restart button, which will reset the grid and score
  • Hitting the Space Bar key on your keyboard can restart the game (mid-game), or escape the instructions page, or generate new tiles at the beginning of the game if you're not happy with the tiles.
  • To return to the welcome page, click the Back to Welcome Page button.

Enjoy playing and good luck reaching 2048!
