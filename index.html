How to Play 2048

1. Click to select a tile, and then use the arrow keys [up, down, left, right] to merge it in the direction of an adjacent tile with a matching number to achieve a higher score.

2. If the number doesnâ€™t match the direction the user selects, the tile trying to merge will shake left and right to signal to the user that the move is not possible.

3. If the tile is eligible to merge, the tiles will double their displayed value and merge into one

4. If there are tiles beneath where they are merging, the tile will simply display the new value

5. If there are no tiles beneath where they merge, the new combined tile will fall until it hits the floor or lands on top of an existing tile.
